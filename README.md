## ğŸ”¹ System Working
1. User enters transaction details (amount, type, time)
2. Input is processed by trained ML models
3. A fraud risk score is calculated
4. Transaction is classified as **Safe** or **Fraud**
5. Result is displayed to the user
6. Transaction data is stored for analysis

## ğŸ”¹ Machine Learning Models Used
